QUELVO

Private chauffeur booking platform.

QUELVO is a full private chauffeur service and booking platform developed by JCDM Technologies. Passengers book rides directly — airport transfers, hourly, events and city-to-city. Every ride includes Music Ride, the shared in-car music experience.

Booking system

Custom-built HTML/JS multi-form booking with WhatsApp and email submission.

Fleet management

Admin panel backed by Supabase — vehicles, drivers, availability tracking.

Fixed-price quoting

Price logic per ride type and zone — transparent pricing shown before confirmation.

Music Ride integration

Every ride links to a Music Ride session — collaborative in-car playlist via QR.

Platform · Live

Music Ride

Shared music experience for private rides.

A real-time collaborative music queue for private rides. Passengers scan a QR code, join a session and add songs from YouTube — no login, no app download required. Built as a full-stack React / Node.js application with Socket.IO.

Real-time queue

Socket.IO powered — song additions and changes sync instantly across all connected devices.

YouTube integration

Search and play any YouTube video. No Spotify API needed — works without any account.

QR session entry

Driver generates a QR at ride start. Passengers join in seconds with any phone browser.

Driver controls

Driver account manages the session — can approve queue, skip, pause and end the session.

React Node.js Socket.IO YouTube IFrame API Railway PostgreSQL JWT auth
